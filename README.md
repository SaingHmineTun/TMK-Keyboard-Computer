# TMK Keyboard

TMK Keyboard provides native keyboard layouts for typing Shan Unicode text.

## Projects

- [TMK-Keyboard-Windows](TMK-Keyboard-Windows/README.md) - Windows MKLC source, build instructions, installer placeholder, and key-map artwork.
- [TMK Keyboard macOS](TMK%20Keyboard%20macOS/README.md) - standalone macOS keyboard layout project.

## Current Layout

The Windows layout currently has two typing layers:

- Normal
- Shift

The Windows source lives at [TMK Keyboard.klc](TMK-Keyboard-Windows/TMK%20Keyboard.klc).
