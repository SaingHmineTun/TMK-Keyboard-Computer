# TMK Keyboard Linux Key Mapping

This document is the source-of-truth mapping for the Linux `m17n/shn-tmk.mim` input method and the native `xkb/symbols/tmk` layout.

The m17n input method supports every output below. The XKB layout can emit only one Unicode keysym per key press, so Shift+F, Shift+K, and Shift+L use documented single-codepoint fallbacks in `xkb/symbols/tmk`.
